## Fedora 字体安装
### 安装 Inconsolata 字体
1. 在 /usr/share/fonts/ 目录下新建文件夹
``` shell
$ mkdir /usr/share/fonts/Inconsolata
```
2. 将字体文件复制到新建的目录下
``` shell
cp Inconsolata-Regular.ttf /usr/share/fonts/Inconsolata
```
